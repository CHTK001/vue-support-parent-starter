# 需求文档：语音朗读 + 热点工具（热力图覆盖层）

## 功能范围

### 功能一：语音朗读模式

在「无障碍」设置分组中，新增「语音朗读」开关。

**行为描述：**
- 开关开启时，使用浏览器原生 Web Speech API（`SpeechSynthesis`）朗读页面内容
- 朗读触发时机：用户焦点切换到可交互元素时（`focus` 事件），朗读该元素的 `aria-label`、`title` 或 `textContent`（优先级依次降低）
- 开关关闭时，立即调用 `speechSynthesis.cancel()` 停止朗读，并移除 `focus` 监听
- 不支持 `speechSynthesis` 的浏览器，开关置灰并显示提示

**验收标准：**
- [ ] 系统设置「无障碍」分组出现「语音朗读」开关
- [ ] 开启后，Tab 切换焦点时能听到元素内容被朗读
- [ ] 关闭后，朗读立即停止
- [ ] 配置持久化到 `$storage.configure.voiceReadEnabled`

### 功能二：热点工具（热力图覆盖层）

#### 2.1 系统设置入口

在「DevTools 精简版」分组（仅开发/测试环境显示）中，新增「热点工具」开关（`devHeatmap`）。

**验收标准：**
- [ ] 仅开发/测试环境下，DevTools 分组出现「热点工具」开关
- [ ] 开关状态持久化到 `$storage.configure.devHeatmap`

#### 2.2 可拖动浮层组件（HeatmapOverlay）

参考 `LiteInspector.vue` 风格，新建 `HeatmapOverlay.vue`，挂载在布局根组件 `index.vue` 中（紧邻 `<LiteInspector />`）。

**浮层结构：**
- 仅在 `devHeatmap` 开启时显示
- 可自由拖动（鼠标拖拽）
- 包含一个独立按钮：「热力图展示」（切换热力图覆盖层的显示/隐藏）
- 按钮激活状态有视觉反馈（高亮/选中样式）

**验收标准：**
- [ ] `devHeatmap` 开启后，页面右下角出现可拖动浮层
- [ ] 浮层可自由拖拽到任意位置
- [ ] 点击「热力图展示」按钮，切换热力图覆盖层

#### 2.3 热力图覆盖层

**行为描述：**
- 热力图激活后，全局监听 `click` 事件，记录每次点击的页面坐标（`clientX`、`clientY`）和点击次数
- 使用全屏 `<canvas>`（`position: fixed; inset: 0; pointer-events: none; z-index: 9998`）渲染热力分布
- 渲染算法：每个点击点绘制径向渐变圆（半径 60px），颜色映射：低频→蓝色，中频→绿色→黄色，高频→红色
- 热力图关闭时，清除 canvas 内容并移除 click 监听
- 提供「清除数据」功能（浮层上第二个按钮）

**验收标准：**
- [ ] 点击页面后，热力图 canvas 上出现对应位置的热力渐变
- [ ] 多次点击同一区域，颜色向红色方向加深
- [ ] 关闭热力图后，canvas 清空，监听移除
- [ ] 「清除数据」按钮清空所有点击记录并重绘

---

## 不在范围内

- 热力图数据持久化（刷新后清空，属于后续迭代）
- 服务端上报点击数据
- 移动端触摸事件支持
