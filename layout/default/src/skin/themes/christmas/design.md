# 圣诞节主题设计文档

## 设计理念

营造温馨、欢乐的圣诞氛围，通过红色、绿色、金色的配色和雪花、圣诞树等元素创造节日体验。

## 配色方案

### 主色调

- **圣诞红**: `#DC143C` - 主要强调色
- **常青绿**: `#2D5016` - 次要强调色
- **金色**: `#FFD700` - 装饰色
- **雪白**: `#FFFFFF` - 背景和文字

### 辅助色

- **深红**: `#8B0000` - 悬停状态
- **深绿**: `#1B3A0F` - 阴影
- **银色**: `#C0C0C0` - 边框和图标
- **冰蓝**: `#E0F2FE` - 背景渐变

## 视觉元素

### 图标

- 雪花 (snowflake)
- 圣诞树 (christmas-tree)
- 礼物盒 (gift-box)
- 彩灯 (lights)
- 铃铛 (bell)

### 动画效果

- 飘雪动画
- 闪烁彩灯效果
- 礼物盒摇摆
- 圣诞树装饰闪烁

## 组件样式

### 按钮

- 圆角: 12px
- 阴影: 0 4px 12px rgba(220, 20, 60, 0.3)
- 悬停: 轻微缩放 + 阴影增强

### 卡片

- 背景: 白色/浅蓝渐变
- 边框: 1px solid 金色
- 阴影: 柔和红色光晕

### 输入框

- 边框: 圣诞红
- 焦点: 金色光晕
- 背景: 白色

## 响应式设计

- 移动端: 减少雪花数量
- 平板: 适中动画效果
- 桌面: 完整雪花和彩灯效果

## 暗黑模式适配

- 背景: 深蓝色调
- 保持主题色彩
- 增强金色和银色对比
