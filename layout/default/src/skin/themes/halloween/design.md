# 万圣节主题设计文档

## 设计理念
营造神秘、欢乐的万圣节氛围，通过橙色、黑色、紫色的配色和南瓜、蝙蝠、幽灵等元素创造节日体验。

## 配色方案

### 主色调
- **南瓜橙**: `#FF6B35` - 主要强调色
- **深夜黑**: `#1A1A2E` - 背景色
- **神秘紫**: `#8B5CF6` - 次要强调色
- **幽灵白**: `#F8F9FA` - 文字和高亮

### 辅助色
- **暗橙**: `#E85D04` - 悬停状态
- **深紫**: `#6D28D9` - 阴影和边框
- **灰雾**: `#4A4A5E` - 禁用状态

## 视觉元素

### 图标
- 南瓜灯 (pumpkin)
- 蝙蝠 (bat)
- 幽灵 (ghost)
- 蜘蛛网 (spider-web)
- 糖果 (candy)

### 动画效果
- 飘落蝙蝠动画
- 闪烁南瓜灯效果
- 漂浮幽灵动画
- 蜘蛛网摇摆效果

## 组件样式

### 按钮
- 圆角: 8px
- 阴影: 0 4px 12px rgba(255, 107, 53, 0.3)
- 悬停: 轻微上浮 + 阴影增强

### 卡片
- 背景: 半透明黑色
- 边框: 1px solid 神秘紫
- 阴影: 柔和紫色光晕

### 输入框
- 边框: 南瓜橙
- 焦点: 紫色光晕
- 背景: 深色半透明

## 响应式设计
- 移动端: 简化动画效果
- 平板: 保留主要动画
- 桌面: 完整动画体验

## 暗黑模式适配
- 自动调整亮度
- 保持主题色调
- 增强对比度
