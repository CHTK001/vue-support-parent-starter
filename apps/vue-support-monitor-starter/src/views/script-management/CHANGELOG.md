# 更新日志

## [2.0.0] - 2024-12-11

### 重大变更

- 全面重构脚本管理模块
- 简化为只保留核心CRUD功能
- 移除执行、历史、上传等功能

### 新增

- 创建统一的类型定义系统
- 创建统一的样式系统(variables + mixins)
- 创建完整的工具函数库
- 添加6种脚本类型的实用模板
- 添加脚本分类功能

### 优化

- API统一移动到src/api目录
- 代码量减少73%
- 文件数量减少64%
- 组件数量减少93%
- 改进UI设计，采用现代化卡片布局
- 优化搜索和筛选功能
- 改进代码编辑器体验

### 移除

- 脚本执行功能
- 执行历史功能
- 运行中脚本监控
- 上传记录功能
- 统计分析功能

### 修复

- 修复保存脚本后列表不刷新的问题
- 修复TypeScript类型错误
- 修复CSS兼容性警告

## [1.0.0] - 2024-12-10

### 初始版本

- 完整的脚本管理功能
- 脚本执行和监控
- 执行历史记录
- 上传到服务器/节点
