# 组件库更新 - 科技风格扩展

## 更新概述

本次更新对组件库进行了全面的科技风格扩展，重构和新增了多个组件，使其完整支持 Element Plus 原生功能和科技风格主题。

**更新日期**: 2025-12-03  
**版本**: 2.0.0  
**开发者**: CH

## 更新内容

### 1. ScButton - 完整封装 el-button ✨

#### 更新说明

- 完整封装 Element Plus 的 el-button 所有功能
- 支持 tech 科技主题和 default 默认主题双模式
- tech 模式下保留科技风格特效，default 模式下使用原生 el-button

#### 新增 Props

| 参数            | 说明                     | 类型    | 默认值    |
| --------------- | ------------------------ | ------- | --------- |
| autofocus       | 自动聚焦                 | boolean | false     |
| color           | 自定义按钮颜色           | string  | —         |
| dark            | 暗黑模式                 | boolean | false     |
| autoInsertSpace | 自动在中文字符间插入空格 | boolean | undefined |
| tag             | 自定义元素标签           | string  | button    |
| loadingIcon     | 加载图标                 | string  | —         |
| bg              | 是否显示背景色           | boolean | false     |

#### 使用示例

```vue
<template>
  <!-- Tech 主题 -->
  <ScButton theme="tech" type="primary">科技按钮</ScButton>

  <!-- 默认主题（使用 el-button） -->
  <ScButton type="primary">默认按钮</ScButton>

  <!-- 自定义颜色 -->
  <ScButton color="#ff6b6b">自定义颜色</ScButton>
</template>
```

### 2. ScPanel - 重构封装 el-collapse ♻️

#### 更新说明

- 删除旧的 ScPanel 实现
- 重新封装 Element Plus 的 el-collapse 和 el-collapse-item
- 新增 ScPanelGroup 容器组件
- 支持 tech 科技主题

#### 组件结构

**ScPanelGroup** - 面板组容器

- 封装 el-collapse
- 管理多个面板的展开/收起状态
- 支持手风琴模式

**ScPanel** - 面板项

- 封装 el-collapse-item
- 支持 tech 主题的科技风格装饰
- 6 种主题颜色可选

#### Props

**ScPanelGroup**

| 参数       | 说明           | 类型                | 默认值  |
| ---------- | -------------- | ------------------- | ------- |
| modelValue | 当前激活的面板 | string/number/array | []      |
| accordion  | 是否手风琴模式 | boolean             | false   |
| theme      | 主题风格       | string              | default |

**ScPanel**

| 参数      | 说明                 | 类型          | 默认值  |
| --------- | -------------------- | ------------- | ------- |
| name      | 面板名称（唯一标识） | string/number | —       |
| title     | 面板标题             | string        | —       |
| disabled  | 是否禁用             | boolean       | false   |
| theme     | 主题风格             | string        | default |
| techTheme | Tech 主题颜色        | string        | cyan    |
| icon      | 图标                 | string        | —       |

#### 使用示例

```vue
<template>
  <!-- Tech 主题 -->
  <ScPanelGroup v-model="activeNames" theme="tech">
    <ScPanel
      name="1"
      title="系统监控"
      icon="ri:dashboard-line"
      theme="tech"
      tech-theme="cyan"
    >
      <div>面板内容</div>
    </ScPanel>
    <ScPanel
      name="2"
      title="数据分析"
      icon="ri:bar-chart-line"
      theme="tech"
      tech-theme="blue"
    >
      <div>面板内容</div>
    </ScPanel>
  </ScPanelGroup>

  <!-- 默认主题（使用 el-collapse） -->
  <ScPanelGroup v-model="activeNames">
    <ScPanel name="1" title="面板 1">
      <div>面板内容</div>
    </ScPanel>
  </ScPanelGroup>
</template>
```

### 3. ScDeco - 科技装饰组件 🆕

#### 组件说明

全新的科技装饰组件，提供多种科技风格的装饰效果，可以为任何元素添加科技感。

#### 装饰类型

1. **corner** - 边角装饰
   - SVG 绘制的边角装饰
   - 支持发光效果

2. **scan** - 扫描线
   - 水平扫描线动画
   - 可自定义速度

3. **grid** - 网格背景
   - 科技感网格纹理
   - 可调节网格大小

4. **glow-border** - 发光边框
   - 四边发光边框
   - 支持脉冲动画

5. **data-flow** - 数据流
   - 垂直数据流动画
   - 可调节数量

6. **particle** - 粒子效果
   - 粒子扩散动画
   - 可调节粒子数量

7. **pulse** - 脉冲效果
   - 边框脉冲动画
   - 呼吸效果

8. **combo** - 组合效果
   - 包含多种装饰的组合
   - 完整的科技风格

#### Props

| 参数          | 说明         | 类型    | 默认值 |
| ------------- | ------------ | ------- | ------ |
| type          | 装饰类型     | string  | corner |
| theme         | 主题颜色     | string  | cyan   |
| size          | 装饰大小     | number  | 20     |
| animated      | 是否启用动画 | boolean | true   |
| gridSize      | 网格大小     | number  | 20     |
| dataFlowCount | 数据流数量   | number  | 5      |
| particleCount | 粒子数量     | number  | 20     |
| color         | 自定义颜色   | string  | —      |
| glowIntensity | 发光强度     | number  | 0.5    |

#### 使用示例

```vue
<template>
  <div style="position: relative; padding: 20px;">
    <!-- 边角装饰 -->
    <ScDeco type="corner" theme="cyan" />

    <!-- 组合装饰 -->
    <ScDeco type="combo" theme="blue" :animated="true" />

    <!-- 粒子效果 -->
    <ScDeco type="particle" theme="green" :particle-count="30" />

    <div>你的内容</div>
  </div>
</template>
```

### 4. ScHeader - 头部组件 🆕

#### 组件说明

封装 el-header 并支持科技风格主题的头部组件。

#### 特性

- 完整封装 el-header 功能
- 支持 tech 科技主题
- 6 种主题颜色
- 支持固定定位
- 支持透明背景
- 网格背景和扫描线动画
- 数据流装饰
- Logo 和标题显示
- 左中右三栏布局

#### Props

| 参数         | 说明           | 类型    | 默认值  |
| ------------ | -------------- | ------- | ------- |
| height       | 头部高度       | string  | 60px    |
| theme        | 主题风格       | string  | default |
| techTheme    | Tech 主题颜色  | string  | cyan    |
| logo         | Logo 图片地址  | string  | —       |
| title        | 标题           | string  | —       |
| icon         | 图标           | string  | —       |
| fixed        | 是否固定       | boolean | false   |
| transparent  | 是否透明       | boolean | false   |
| showDataFlow | 是否显示数据流 | boolean | true    |
| blur         | 背景模糊       | boolean | true    |

#### 插槽

- **left** - 左侧内容
- **center** - 中间内容
- **right** - 右侧内容
- **logo** - Logo 内容
- **default** - 默认内容

#### 使用示例

```vue
<template>
  <!-- Tech 主题 -->
  <ScHeader
    theme="tech"
    tech-theme="cyan"
    title="系统管理平台"
    icon="ri:dashboard-line"
    :fixed="true"
  >
    <template #right>
      <ScButton theme="tech" size="small">退出</ScButton>
    </template>
  </ScHeader>

  <!-- 默认主题 -->
  <ScHeader height="60px">
    <div>头部内容</div>
  </ScHeader>
</template>
```

## 主题颜色系统

所有科技风格组件支持以下 6 种主题颜色：

### Cyan（青色）- 默认

- 主色: `#00f6ff`
- 适用场景: 数据监控、系统管理

### Blue（蓝色）

- 主色: `#1890ff`
- 适用场景: 数据分析、商业应用

### Green（绿色）

- 主色: `#00ff88`
- 适用场景: 健康状态、成功提示

### Purple（紫色）

- 主色: `#b37feb`
- 适用场景: 安全防护、高级功能

### Orange（橙色）

- 主色: `#ff9500`
- 适用场景: 警告提示、重要通知

### Red（红色）

- 主色: `#ff4d4f`
- 适用场景: 错误提示、危险操作

## 设计规范

### 视觉特效

1. **网格背景**
   - 网格大小: 20px × 20px
   - 透明度: 0.05 - 0.1
   - 颜色: 主题色

2. **发光效果**
   - 使用 box-shadow 和 filter
   - 发光半径: 5-15px
   - 颜色: 主题色 + 透明度

3. **边角装饰**
   - SVG 绘制
   - 支持动画效果
   - 带发光阴影

4. **扫描线**
   - 水平或垂直移动
   - 渐变效果
   - 循环动画

### 动画规范

- 过渡时间: 0.3s
- 缓动函数: cubic-bezier(0.4, 0, 0.2, 1)
- 扫描线: 3-4s linear infinite
- 脉冲: 2s ease-in-out infinite

## 使用建议

### 1. 主题选择

- 根据应用场景选择合适的主题颜色
- 保持整个应用主题色调一致
- 避免在同一页面使用过多不同颜色

### 2. 性能优化

- 大量组件时考虑关闭部分动画
- 使用 `animated="false"` 禁用不必要的动画
- 避免嵌套过多装饰组件

### 3. 可访问性

- 确保文字对比度符合标准
- 为图标按钮添加 aria-label
- 支持键盘导航

### 4. 响应式设计

- 在移动端适当调整组件尺寸
- 考虑触摸设备的交互体验
- 使用媒体查询优化显示
