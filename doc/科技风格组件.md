# 科技风格组件开发文档

## 概述

本文档记录了为组件库添加科技风格（Tech Theme）支持的完整实现过程。科技风格采用赛博朋克/未来科技的视觉设计，具有发光边框、网格背景、扫描线动画等特效。

**开发时间**: 2025-12-03  
**开发者**: CH  
**版本**: 1.0.0

## 实现的组件

### 1. ScCard - Tech 布局

**文件位置**: `packages/components/ScCard/layouts/Tech.vue`

#### 特性

- 支持 6 种主题颜色：cyan（青色）、blue（蓝色）、green（绿色）、purple（紫色）、orange（橙色）、red（红色）
- 网格背景效果
- 扫描线动画
- SVG 边角装饰
- 发光边框效果
- 数据流动画（可选）
- 悬停和激活状态动画

#### 使用方法

```vue
<template>
  <ScCard
    layout="tech"
    title="系统监控"
    icon="ri:dashboard-line"
    theme="cyan"
    :active="true"
  >
    <div>卡片内容</div>
    <template #footer>
      <button>操作按钮</button>
    </template>
  </ScCard>
</template>
```

#### Props

| 参数         | 说明               | 类型    | 可选值                            | 默认值 |
| ------------ | ------------------ | ------- | --------------------------------- | ------ |
| title        | 卡片标题           | string  | —                                 | —      |
| icon         | 图标               | string  | —                                 | —      |
| showHeader   | 是否显示头部       | boolean | —                                 | true   |
| hoverable    | 是否可悬停         | boolean | —                                 | true   |
| active       | 是否激活状态       | boolean | —                                 | false  |
| theme        | 主题色             | string  | cyan/blue/green/purple/orange/red | cyan   |
| showDataFlow | 是否显示数据流动画 | boolean | —                                 | true   |
| padding      | 内边距             | string  | —                                 | 20px   |

### 2. ScDialog - Tech 主题

**文件位置**: `packages/components/ScDialog/src/index.vue`

#### 特性

- 深色半透明背景
- 发光边框效果
- 网格背景纹理
- 渐变边角装饰
- 自定义滚动条样式
- 按钮悬停发光效果
- 支持拖拽、缩放、最小化等功能

#### 使用方法

```vue
<template>
  <ScDialog
    v-model="visible"
    title="系统配置"
    theme="tech"
    mode="custom"
    width="600px"
    :resizable="true"
  >
    <div>对话框内容</div>
    <template #footer>
      <button>取消</button>
      <button>确定</button>
    </template>
  </ScDialog>
</template>
```

#### Props

在原有 ScDialog 的基础上新增:

| 参数  | 说明     | 类型   | 可选值       | 默认值  |
| ----- | -------- | ------ | ------------ | ------- |
| theme | 主题风格 | string | default/tech | default |

### 3. ScButton - 科技按钮

**文件位置**: `packages/components/ScButton/index.vue`

#### 特性

- 支持多种按钮类型和尺寸
- 科技风格边角装饰
- 发光悬停效果
- 加载状态动画
- 支持图标按钮
- 支持圆形、圆角、文字、链接等样式

#### 使用方法

```vue
<template>
  <!-- 基础按钮 -->
  <ScButton theme="tech">默认按钮</ScButton>
  <ScButton theme="tech" type="primary">主要按钮</ScButton>

  <!-- 图标按钮 -->
  <ScButton theme="tech" icon="ep:search">搜索</ScButton>
  <ScButton theme="tech" circle icon="ep:plus" />

  <!-- 加载状态 -->
  <ScButton theme="tech" loading type="primary">加载中</ScButton>

  <!-- 禁用状态 -->
  <ScButton theme="tech" disabled>禁用按钮</ScButton>
</template>
```

#### Props

| 参数       | 说明           | 类型    | 可选值                                      | 默认值  |
| ---------- | -------------- | ------- | ------------------------------------------- | ------- |
| type       | 按钮类型       | string  | default/primary/success/warning/danger/info | default |
| size       | 按钮尺寸       | string  | large/default/small                         | default |
| theme      | 主题风格       | string  | default/tech                                | default |
| disabled   | 是否禁用       | boolean | —                                           | false   |
| loading    | 是否加载中     | boolean | —                                           | false   |
| plain      | 是否朴素按钮   | boolean | —                                           | false   |
| round      | 是否圆角按钮   | boolean | —                                           | false   |
| circle     | 是否圆形按钮   | boolean | —                                           | false   |
| text       | 是否文字按钮   | boolean | —                                           | false   |
| link       | 是否链接按钮   | boolean | —                                           | false   |
| icon       | 图标           | string  | —                                           | —       |
| suffixIcon | 后置图标       | string  | —                                           | —       |
| nativeType | 原生 type 属性 | string  | button/submit/reset                         | button  |

#### Green（绿色）

- Secondary: `#00cc6a`

#### Purple（紫色）

### 动画效果

1. **扫描线动画**
   - 持续时间: 3s
   - 动画类型: linear infinite
   - 效果: 从上到下扫描

2. **边角脉冲动画**
   - 持续时间: 2s
   - 动画类型: ease-in-out infinite
   - 效果: 缩放和透明度变化

3. **发光动画**
   - 持续时间: 2s
   - 动画类型: ease-in-out infinite
   - 效果: 阴影强度变化

4. **数据流动画**
   - 持续时间: 2s
   - 动画类型: linear infinite
   - 效果: 点从上到下移动

### 视觉特效

1. **网格背景**
   - 网格大小: 20px × 20px
   - 线条颜色: 主题色 + 透明度
   - 整体透明度: 0.05 - 0.3

2. **发光边框**
   - 边框宽度: 1-2px
   - 发光半径: 5-15px
   - 颜色: 主题色 + 透明度

3. **边角装饰**
   - 使用 SVG 绘制
   - 支持悬停缩放效果
   - 带发光阴影

## 技术实现

### 关键技术点

1. **CSS 变量**
   - 使用 CSS 变量实现主题切换
   - 便于维护和扩展

2. **backdrop-filter**
   - 实现毛玻璃效果
   - 增强视觉层次感

3. **box-shadow**
   - 多层阴影叠加
   - 实现发光效果

4. **CSS 动画**
   - @keyframes 定义动画
   - 使用 cubic-bezier 缓动函数

5. **SVG 图形**
   - 绘制边角装饰
   - 支持动态样式

### 浏览器兼容性

- Chrome 88+
- Firefox 94+
- Safari 15.4+
- Edge 88+

**注意**: backdrop-filter 在某些浏览器中可能需要前缀或不支持。

## 使用建议

1. **主题选择**
   - Cyan: 适合数据监控、系统管理
   - Blue: 适合数据分析、商业应用
   - Green: 适合健康状态、成功提示
   - Purple: 适合安全防护、高级功能
   - Orange: 适合警告提示、重要通知
   - Red: 适合错误提示、危险操作

2. **性能优化**
   - 避免在同一页面使用过多动画效果
   - 大量卡片时可以关闭 `showDataFlow`
   - 使用 `will-change` 优化动画性能

3. **可访问性**
   - 确保文字对比度符合 WCAG 标准
   - 为图标按钮添加 aria-label
   - 支持键盘导航
